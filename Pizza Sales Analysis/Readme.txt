This repository contains a Power BI dashboard and analysis of pizza sales for a business.
The analysis is designed to assist in decision-making and optimizing sales and operations.

USAGE

To explore the Pizza Sales Dashboard, follow these steps:

#1 Download the Power BI Desktop software from the official Power BI website.
#2 Open the project in Power BI Desktop by opening the Pizza_Sales_Dashboard.pbix file.
#3 Interact with the dashboard to analyze pizza sales data, gain insights, and make data-driven decisions.

Explore the Pizza Sales Dashboard to gain insights into pizza sales trends and make data-driven decisions for your business. 
If you have any questions or suggestions, please feel free to reach out.
